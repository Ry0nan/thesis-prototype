# Kamias Defect Detection

This project implements a data-efficient semi-supervised learning framework for detecting surface defects in Kamias using YOLOv8 and ResNet18.

Pipeline:
1. YOLOv8 for object detection
2. ROI cropping
3. ResNet18 for classification
4. Semi-supervised learning using pseudo-labeling

Status: Prototype phase (dataset collection ongoing)
